# מערכת תמלול שיעורי תורה

אפליקציה פשוטה לתמלול אוטומטי של שיעורי תורה באמצעות Google Gemini API.

## תכונות עיקריות

- תמלול קבצי MP3 באמצעות מודל השפה המתקדם של Google Gemini
- חלוקה אוטומטית של קבצים ארוכים למקטעים קטנים יותר לתמלול מיטבי
- ממשק משתמש נוח בעברית עם כל ההוראות הנדרשות
- אפשרות להתאמה אישית של הפרומפט לתמלול
- ניהול מתקדם של מכסות טוקנים במספר פרויקטים של Google
- הצגת התקדמות התמלול בזמן אמת
- אפשרות להורדת התמלול המלא בסיום

## הוראות שימוש

### 1. הכנות מקדימות

לפני השימוש באפליקציה, תצטרך:
1. **מפתח API של Google AI Studio** - ניתן לקבל בחינם דרך [Google AI Studio](https://aistudio.google.com/)
2. **מזהי פרויקטים** - ניתן ליצור פרויקטים חדשים ב-Google Cloud לניהול מכסות הטוקנים החינמיות

### 2. תהליך התמלול

1. הזן את מפתח ה-API שלך ופרטי הפרויקטים בסרגל הצד
2. העלה קובץ MP3 של השיעור שברצונך לתמלל
3. (אופציונלי) התאם את הפרומפט לתמלול לצרכים שלך
4. לחץ על כפתור "תמלל" והמתן להשלמת התהליך
5. לאחר סיום התמלול, תוכל לראות את התוצאה ולהוריד אותה כקובץ טקסט

### 3. התאמת הפרומפט

מומלץ להשתמש בפרומפט ברירת המחדל שמותאם במיוחד לתמלול שיעורי תורה.
אם יש לך דרישות מיוחדות, ניתן להתאים את הפרומפט:

1. לחץ על "פרומפט מותאם אישית לתמלול" כדי לפתוח את אזור העריכה
2. ערוך את הטקסט לפי הצורך
3. הפרומפט המותאם יישלח למודל ה-AI בזמן התמלול

## דרישות מערכת

האפליקציה פועלת ב-Streamlit ודורשת:
- Python 3.8 או גרסה חדשה יותר
- FFmpeg (לעיבוד קבצי אודיו)
- חיבור אינטרנט לשירותי Google

## התקנה מקומית (למי שמעוניין להריץ על המחשב האישי)

1. התקן את הספריות הנדרשות:
   ```
   pip install -r requirements.txt
   ```

2. התקן FFmpeg (נדרש לעיבוד אודיו):
   - **Windows**: הורד מ-[ffmpeg.org](https://ffmpeg.org/download.html) והוסף לנתיב המערכת
   - **Mac**: `brew install ffmpeg`
   - **Linux**: `sudo apt install ffmpeg`

3. הרץ את האפליקציה:
   ```
   streamlit run transcription_app.py
   ```

## שאלות נפוצות

### מה לעשות אם מוצגת שגיאת API?
ודא שמפתח ה-API שהזנת תקין וכי לא הגעת למגבלת הטוקנים היומית. אם הבעיה נמשכת, נסה להשתמש בפרויקט אחר.

### כמה זמן לוקח לתמלל שיעור?
זמן התמלול תלוי באורך השיעור, איכות ההקלטה, ומהירות החיבור. שיעור של שעה עשוי לקחת בין 15-30 דקות לתמלול מלא.

### איך אני משיג מפתח API לשירות?
1. היכנס ל-[Google AI Studio](https://aistudio.google.com/)
2. הירשם/התחבר עם חשבון Google
3. לחץ על "Get API key" בדף הבית
4. צור מפתח API חדש ועתק אותו לאפליקציה

### יש מגבלה על כמות התמלולים?
שירותי Google Gemini כוללים מכסת טוקנים חינמית יומית. האפליקציה מנהלת את השימוש בטוקנים ומאפשרת לך להשתמש במספר פרויקטים כדי להגדיל את המכסה הזמינה.

### האם ניתן לתמלל בשפות אחרות מלבד עברית?
כן, עליך להתאים את הפרומפט לשפה הרצויה. המערכת תומכת בכל השפות שמודל Gemini תומך בהן.
